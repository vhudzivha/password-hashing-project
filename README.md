# Password Hashing Project

## 👨‍💻 Student Details
Name: Sunnyboy  
Project: Password Hashing Simulation  

## 📌 Description
This project demonstrates how password hashing works using PHP and MySQL.

Passwords are not stored in plain text. They are hashed using PHP functions to improve security.

## ⚙️ Tools Used
- PHP
- MySQL (phpMyAdmin)
- XAMPP
- HTML/CSS

## 🔐 Features
- User Registration (Password Hashing using password_hash())
- User Login (Password Verification using password_verify())

## 🚀 How to Run
1. Start XAMPP (Apache & MySQL)
2. Open phpMyAdmin
3. Create database: password_db
4. Import table structure
5. Run project:
   http://localhost/password_project/register.php

## 🧠 Concept
- Password hashing protects user data
- Even if database is hacked, passwords cannot be read
